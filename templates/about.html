About this project

Blue Oak Express Bank is a demonstration build

What this is

Blue Oak Express Bank is a fictional institution built to demonstrate a complete retail banking interface: authentication, account overview, transfers between accounts, and a full transaction ledger. It is not a bank, it holds no money, and it is not affiliated with any real financial institution. Every account, balance, card and transfer you can see here is invented test data.

How it works

The application runs on Flask with server-rendered Jinja templates. Accounts and transactions persist to flat JSON files rather than a database, which keeps the project easy to clone and run without setup. Passwords are stored as Werkzeug scrypt hashes. Transfers debit the sender, credit the recipient, and append an immutable entry to the ledger; when a recipient cannot be matched, the transfer is held for review rather than silently completed.

Design

The interface takes its cues from the visual language of traditional banking rather than contemporary fintech — engine-turned guilloché line work of the kind engraved on banknotes and share certificates, statement rules, and tabular figures set in a monospaced face so that columns of numbers align the way they do on a printed statement. The type pairs Spectral for display with IBM Plex Sans and IBM Plex Mono.

Running it locally

Install the requirements, then run python app.py and open localhost:5000. Seeded accounts live in users.json; create_users.py will regenerate them with fresh password hashes.
